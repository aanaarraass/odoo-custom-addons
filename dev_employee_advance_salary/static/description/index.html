Employee Advance Salary
Manage employee advance salary request with manager approvals in odoo.

DevIntelle Consulting Service Pvt.Ltd.

Features
  • Employee advance salary request.
  • Employee Create Own Advance Salay.
  • Advance Salary User can approve the request.
  • HR Officer confirm the request.
  • Integrated with accounting for payment.
  • Integrated with HR Payroll & Employee Payslips.
  • Employee advance salary notification to Employee & HR Manager
  • Two Way Approval Process Advance Salary Request
  • Employee advance salary limit & how many Salary request validation
  • Draft->Request->Approved by Department Manager->Confirm by HR Manager->Paid.

Setup & Configure

Salary Limit & Request Per Year

In Employee Form fill Salary limit and How many request Employee create per year.

Warning to define the Salary limit and per year Request

If Create directly Advance Salary Request without define salary limit and per year request then System will give you validation Error for define the Salary limit and request limit per year.

Request Amount over the Salary limit Amount

If Request amount is greater then the Salary limit amount of the employee then System will give you this validation Warning.

Some Request in Process

If some Advance Salary Request will be in ongoing process then you will not create other request.

Advance Salary Request limit warning

if your advance salary request is more then advance salary request per year then system will give this waning.

Advance Salary Request

Advance Salary Request Form fill by the employee and send the request to the Department manager.

Advance Salary Request Send to the Department Manager

When Employee Click the Send Request button in Advance Salary Request form then one email send to the Department Manager and request from goes to the Requst State.

Approve Addvance Salary Request

After seen the email the department manager login, view and then after he will Make approve or Reject the Advance Salary Request.

Approved Addvance Salary Request By Department Manager

Department manager Click to Approve Request button then Email Send to the Hr Manager for the Confirm the Employee Advance Salary Request.

Reject Addvance Salary Request By Department Manager

Department manager Click to Reject Request button then Email Send to the Employee For Reject the Advance Salary.

Confirm the Employee Advance Salary Request by HR Manager

After the Getting mail the HR manager confirm or reject the advance salary request.

Advance Salary Confirmation Mail

Clicking the Confirm Button in Advance Salary Request Form by HR Manager Employee Get the Email For Advance Salary Request is Confirmed.

Paid the Advance Salary

After Confirmed Request Hr manager fill the other values and make paid the Advance Salary by clicking the Paid button.

Advance Salary Paid

By cliking the paid button from the Salary request from Payment is created and Salary Request is goes to Paid State.

Advance Salary Report

Advance Salary Report.

Advance Salary Deduct From the Payslip

After Paid the Advance Salary after then Payslip is create that time Advance Salary Amount Will be deduct from the Employee Salary.

Advance Salary - Payslip

Advance Salary Amount Will deduct from the Employee Salary Amount

View Employee Advance Salary From Employee Form

From the Employee Form You can see employee All Advance Salary Request by clicking the Salary Request Button.
